Website: A website is a collection of related web pages and content that are accessible through the internet and typically hosted on a single domain. In other words,a collection of files accessed through a web address, covering a particular theme or subject, and managed by a particular person or organization.eg.-facebook,youtube,instagram,etc.

WWW: "www" stands for "World Wide Web." It is a common prefix used in the domain names of websites to identify them as part of the World Wide Web. The WWW is a global network of information and resources that are linked together via hyperlinks.

HTTP/HTTPs: HTTP stands for Hypertext Transfer Protocol, is a fundamental protocol used on the internet for communication between web browsers and web servers. It is the foundation of data communication in the World Wide Web.
Hypertext transfer protocol secure (HTTPS) is the secure version of HTTP.It is encrypted in order to increase security of data transfer.

Client-Server: Client-server is a model where computing tasks and services are divided between two distinct entities: the client and the server. These entities communicate with each other over a network to fulfill specific tasks and provide various services.

Domain: When referring to an Internet address or name, a domain or domain name is the location of a website.

Host: Companies providing web hosting services are sometimes called web hosts.

Hosting: hosting is an online service that makes your website’s content accessible on the internet. When you purchase a hosting plan, you are renting space on a physical server to store all the website’s files and data.

FTP: The File Transfer Protocol(FTP) is a standard communication protocol used for the transfer of computer files from a server to a client on a computer network.

Path: A path is a string of characters used to uniquely identify a location in a directory structure.

Extension: A file extension or file name extension is the ending of a file that helps identify the type of file in operating systems, such as Microsoft Windows.

Editor/vscode: Visual Studio Code, also known as VS Code is a source-code editor made by Microsoft with the Electron Framework, for Windows, Linux and macOS. with support for development operations like debugging, task running, syntax highlighting & intelligent code completion.

URL-URI/URN: A URI (Uniform Resource Identifier) is a formal system for uniquely identifying resources and consists of two types: URLs (Uniform Resource Locator) and URNs (Uniform Resource Name).A URL (Uniform Resource Locator) is a unique identifier used to locate a resource on the Internet. It is also referred to as a web address. URLs consist of multiple parts including a protocol and domain name that tell a web browser how and where to get a resource.

Address bar: The address bar is that displays the name or the URL (uniform resource locator) of the current web page. Users request websites and pages by typing either the name or the URL into the address bar.

Title bar: The title bar is a horizontal bar located at the top of a window in a GUI. It displays the title of the software, name of the current document or file.
