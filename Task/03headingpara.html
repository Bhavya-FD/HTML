Tags vs Elements vs Attributes

Tags:HTML tags are like keywords.90% of tags content both start & end tag wether 10% tags does not have any end tag,they are called self closing tag/void tag/Empty tag.
Elements:An HTML element is defined by a start tag, some content, and an end tag.
Attributes:HTML attributes provide additional information about HTML elements.
